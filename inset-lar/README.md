# Inset Lar - Landing Page

## Superando as Limitações do WordPress

A **Inset Lar** possuía um site institucional construído em WordPress que eu mesmo havia desenvolvido a anos atrás quando aprendi Wordpress (versão anterior disponível em [insetlar.com.br](https://insetlar.com.br/)). A plataforma antiga trazia problemas graves para o negócio: o site travava constantemente, a navegação era ruim no celular e o SEO técnico era extremamente fraco, prejudicando a captação orgânica de clientes.

Considerando que a empresa precisava basicamente de uma página estática rápida para apresentar seus serviços de dedetização e direcionar o cliente para o WhatsApp, identifiquei a oportunidade perfeita de descartar o CMS pesado e construir uma solução sob medida em React.

## A Solução

Recriei a página inteira focando no que realmente importa: velocidade, estabilidade e conversão.

**Principais Melhorias:**
* **Fim dos Travamentos:** A saída do WordPress para uma aplicação React (construída com Vite) eliminou a lentidão e as quedas do servidor.
* **Arquitetura de Conversão:** Layout redesenhado para guiar o usuário até o contato direto, com os serviços da empresa explicados de forma clara e objetiva.
* **Performance e SEO:** Código semântico e tempo de resposta instantâneo para resolver o problema de indexação que a versão antiga sofria no Google.
* **Design Responsivo:** Interface limpa, profissional e 100% adaptável a smartphones, que é de onde vem a maior parte do tráfego do cliente.

## 🛠 Tecnologias Utilizadas

* **Framework:** React + Vite (para build rápido e otimizado).
* **Estilização:** Tailwind CSS (garantindo padronização visual, responsividade fácil e carregamento leve).
* **Linguagem:** JavaScript / JSX.
