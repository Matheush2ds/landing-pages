# Junin Auto Som - Landing Page

## De WordPress para React

A presença digital da **Junin Auto Som (Unidade Cidade Jardim)** rodava em WordPress (versão antiga disponível em [junincidadejardim.com.br](https://junincidadejardim.com.br/)). O grande problema era que o site travava com frequência, o trabalho de SEO era muito fraco e o layout quebrava em dispositivos móveis. 

Como a necessidade real do cliente era apenas apresentar a loja e captar leads (um site estático), vi uma excelente oportunidade de reconstruir tudo do zero. O objetivo foi eliminar as gambiarras dos plugins WordPress, resolver os travamentos e entregar um código limpo e moderno que realmente transmitisse a autoridade de uma loja de acessórios automotivos.

## Solução e Melhorias Implementadas

Desenvolvi uma **Landing Page de Alta Conversão** focada em resolver o problema de performance e melhorar a experiência do usuário.

* **Design e UI:** Estética Dark Mode usando *Glassmorphism*, paleta de cores fiel à marca (#FFB800) e foco na responsividade real.
* **SEO Técnico Avançado:** Como o SEO anterior era ineficiente, implementei `React Helmet Async` para gestão de meta-tags dinâmicas e dados estruturados (Schema.org), garantindo que o Google indexe a página corretamente.
* **Foco em Conversão (CRO):**
  * Botão flutuante do WhatsApp fixo e CTAs com posicionamento estratégico.
  * Seção focada em resolver dores reais ("Comprou e não sabe instalar?").
* **Confiança:** Carrossel destacando a estrutura física da loja (Fachada e Interior) para gerar credibilidade imediata.
* **Performance:** A migração para Vite substituiu o carregamento lento do banco de dados do WordPress por um *load time* quase instantâneo.

## Tecnologias Utilizadas

* **Core:** React 19, Vite.
* **Estilização:** CSS3 Moderno (Variáveis, Flexbox, Grid), Design Responsivo.
* **Animações:** Framer Motion.
* **Ícones:** Lucide React.
* **SEO:** React Helmet Async.
