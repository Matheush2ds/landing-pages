# Portfólio de Landing Pages e Sites Institucionais

Este repositório atua como um *monorepo* que centraliza diversos projetos de front-end desenvolvidos para clientes reais. O foco de cada entrega é resolver problemas de negócios através de interfaces modernas, com alta performance (SEO e *Load Time*) e conversão.

## Stack Tecnológica Geral

A base tecnológica utilizada nos projetos engloba:

* **Ecossistema React:** React 19, Next.js 15 (App Router), Vite.
* **Linguagens:** TypeScript e JavaScript (ES6+).
* **UI & Estilização:** Tailwind CSS, PostCSS, Framer Motion (animações).
* **Foco:** Mobile-first, SEO Técnico (React Helmet Async, Server-Side Rendering) e Design Responsivo.
