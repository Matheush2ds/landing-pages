# Sawti Engenharia

**Acesse o site em produção:** [sawtiengenharia.com.br](https://sawtiengenharia.com.br/)

## O Contexto

A Sawti Engenharia não possuía presença digital oficial. O desafio foi projetar e desenvolver o primeiro site institucional da empresa do zero, criando uma vitrine profissional para apresentar seus serviços, projetos e estabelecer autoridade no setor de engenharia. 

## A Solução

Desenvolvi uma aplicação web moderna e de alta performance utilizando o ecossistema React. O foco foi entregar uma navegação fluida, um layout que transmita confiança e uma base técnica sólida para futuras expansões do site.

**Principais Destaques:**
* **Arquitetura Moderna:** Construído com **Next.js (App Router)** para garantir renderização otimizada e performance no carregamento inicial.
* **Segurança de Tipagem:** Código desenvolvido em **TypeScript**, garantindo maior previsibilidade, menos bugs em produção e facilitando a manutenção futura.
* **UI/UX Design:** Interface responsiva e estilizada com **Tailwind CSS**, adaptando-se perfeitamente a dispositivos móveis e desktops.
* **Gestão de Conteúdo:** O site foi estruturado para consumir dados de forma dinâmica (projetos e FAQ), permitindo atualizações de conteúdo sem necessidade de alterar o código-fonte principal.

*(Nota de Segurança: Rotas administrativas, scripts de manipulação de arquivos e diretórios de upload de clientes foram intencionalmente omitidos deste repositório público por questões de privacidade).*

## Tecnologias Utilizadas

* **Framework:** Next.js 15 (App Router)
* **Linguagem:** TypeScript
* **Estilização:** Tailwind CSS + PostCSS
* **Deploy/Hospedagem:** Otimizado para rodar em ambientes Node.js / Vercel
